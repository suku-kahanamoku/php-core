{ email, password }

validace: email formát, prázdná pole
chyba → 422

← 200 · { token, id, email, role, name }

POST/orders🔒 requiredvytvoření objednávky (transakce)
POST /orders
{ items:[{product_id, quantity}], currency?, payment_method?, shipping_address_id?, note? }
Auth::require() · items pole povinné, nesmí být prázdné → 422
DB::beginTransaction()
pro každý item: SELECT id,name,price,stock_quantity FROM product WHERE id=? AND status=active AND deleted_at IS NULL
produkt nenalezen / neaktivní → rollback + 422
stock < quantity → rollback + 422 Insufficient stock
vypočítá line total = price × qty · sečte totalAmount
INSERT INTO `order` (order_number, user_id, status=pending, total_amount, currency, payment_method, …)
INSERT INTO order_item (order_id, product_id, quantity, unit_price, total_price) × n
UPDATE product SET stock_quantity = stock_quantity − qty WHERE id=? × n
DB::commit()
← 201 · { id, total_amount }
