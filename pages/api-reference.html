🔑 Auth  /api/auth/…

POST /auth/login 🔓 public Přihlášení – vrátí Bearer token ▼
Request body (JSON) 
{
  "email": "admin@example.com",
  "password": "password"
}
Response 200 
{
  "success": true,
  "data": {
    "token": "a3f9c2d8...",
    "id": 1,
    "email": "admin@example.com",
    "role": "admin"
  }
}
curl
curl -X POST $BASE/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email":"admin@example.com",
    "password":"password"
  }'

🚫 Chybové HTTP kódy

200OK – úspěch
201Created – nová entita vytvořena
400Bad Request – chybná data
401Unauthorized – chybí / expirovaný token
403Forbidden – špatná role / neznámý host
404Not Found – entita neexistuje
409Conflict – duplikát / FK constraint
422Unprocessable – validační chyba
500Server Error – neočekávaná chyba
